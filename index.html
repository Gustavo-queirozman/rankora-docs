Rankora – Official Documentation

AI-powered content generator with built-in SEO for WordPress.

Rankora is a premium WordPress plugin that helps you generate high-quality, SEO-optimized content using artificial intelligence directly inside your WordPress dashboard.

Important: Rankora requires a valid license key to function. Without activation, all premium features remain locked.

1. Installation

  1. Log in to your WordPress Admin Dashboard.
  2. Go to Plugins → Add New.
  3. Click Upload Plugin and select rankora2_gumroad.zip.
  4. Click Install Now and then Activate.
    5. Access -> "Tools" -> Rankora Licenseli>
  5. Access -> "Settings" -> Rankora -> Enter the OPENIA API key

2. License Activation (Required)

After installing the plugin, you must activate it using the license key provided at the time of purchase.

  1. After activation, open Rankora from the WordPress admin menu.
  2. Navigate to the License or Settings section.
  3. Paste your License Key received from Gumroad.
  4. Click Activate License.
License Rules:
  • Each license is valid for one WordPress installation.
  • An active subscription is required to use the plugin.
  • If the subscription is canceled or expires, access to premium features will be disabled.

3. Configuration

  1. Open Rankora from the WordPress admin menu.
  2. Enter your OpenAI API Key.
  3. Select the default content language.
  4. Choose the desired content type:
    • TOFU (Top of Funnel)
    • MOFU (Middle of Funnel)
    • BOFU (Bottom of Funnel)
  5. Configure SEO options.
  6. Click Save Settings.

4. How to Use

  1. Navigate to Rankora → Generate Content.
  2. Enter a topic or target keyword.
  3. Select the desired content type.
  4. Click Generate Post.
  5. Review the generated content.
  6. Edit if needed and publish.

5. Requirements

  • WordPress 6.0 or higher
  • PHP 8.0 or higher
  • Valid OpenAI API Key
  • Active internet connection
  • Valid and active Rankora license

6. API Usage Limits

API usage depends on your OpenAI subscription plan and rate limits. Rankora does not store generated content unless you explicitly save or publish it.

7. Support & Updates

Support is provided through the official support channel or the marketplace where the plugin was purchased.

Plugin updates are released regularly. An active license is required to receive updates and continued access to features.

8. Frequently Asked Questions (FAQ)

What happens if I do not activate my license?

Without license activation, Rankora will not allow content generation or access to premium features.

Can I use one license on multiple websites?

No. Each license is limited to a single WordPress installation.

Does Rankora replace human writers?

No. Rankora is an assistant tool. Human review is strongly recommended.

Is Rankora GDPR compliant?

Yes. Rankora does not store personal data. All actions are user-initiated.
