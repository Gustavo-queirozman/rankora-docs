Rankora – Official Documentation

AI-powered content generator with built-in SEO for WordPress.

Rankora helps you generate high-quality, SEO-optimized content using artificial intelligence, directly inside your WordPress dashboard.

1. Installation

  1. Log in to your WordPress Admin Dashboard.
  2. Navigate to Plugins → Add New.
  3. Click Upload Plugin and select rankora2_gumroad.zip.
  4. Click Install Now and then Activate.
Note: You must have a valid license key to use the plugin features.

2. Configuration

  1. Open Rankora from the WordPress admin menu.
  2. Enter your OpenAI API Key.
  3. Select the default language.
  4. Configure content type options:
    • TOFU (Top of Funnel)
    • MOFU (Middle of Funnel)
    • BOFU (Bottom of Funnel)
  5. Adjust SEO settings as needed.
  6. Click Save Settings.

3. How to Use

  1. Go to Rankora → Generate Content.
  2. Enter a topic or target keyword.
  3. Select the desired content type (TOFU, MOFU, or BOFU).
  4. Click Generate Post.
  5. Review the generated content.
  6. Edit if necessary and publish.

4. Requirements

  • WordPress version 6.0 or higher
  • PHP version 8.0 or higher
  • Valid OpenAI API Key
  • Active internet connection
  • Valid Rankora license

5. API Usage Limits

API usage depends entirely on your OpenAI plan and rate limits. Rankora does not store or reuse generated content unless you explicitly save or publish it.

6. Support & Updates

Support is provided through the official support channel or marketplace comments, depending on where the plugin was purchased.

Updates are released regularly and include new features, improvements, and bug fixes. Active licenses are required to receive updates.

7. Frequently Asked Questions (FAQ)

Does Rankora replace human writers?

No. Rankora is a productivity tool. Human review and editing are strongly recommended before publishing content.

Is Rankora GDPR compliant?

Yes. Rankora does not store personal user data. All content generation is initiated by the user.
