Rankora – Official Documentation

AI-powered content generator with built-in SEO for WordPress.

1. Installation

  1. Log in to your WordPress Admin Dashboard.
  2. Go to Plugins → Add New.
  3. Click Upload Plugin and select rankora.zip.
  4. Install and activate the plugin.

2. Configuration

  1. Open Rankora from the WordPress menu.
  2. Enter your OpenAI API Key.
  3. Configure language, content type (TOFU, MOFU, BOFU) and SEO options.
  4. Save the settings.

3. How to Use

  1. Navigate to Rankora → Generate Content.
  2. Enter a topic or keyword.
  3. Select the desired content type.
  4. Click Generate Post.
  5. Review and publish.

4. Requirements

  • WordPress 6.0+
  • PHP 8.0+
  • Valid OpenAI API Key
  • Internet connection

5. API Usage Limits

API usage depends on your OpenAI plan. Rankora does not store generated content without user action.

6. Support & Updates

Support is provided via Envato comments or official support channel. Updates are released regularly.

7. FAQ

Does Rankora replace human writers?
No. Human review is recommended.

Is Rankora GDPR compliant?
Yes. No personal data is stored.
